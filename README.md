## 설치 방법

`resourcepacks` 에 폴더를 위치시킨 후 리소스팩을 활성화하세요.

OS 별 디렉토리는 아래를 참조

### Windows
```
%APPDATA%\.minecraft\resourcepacks
```

### Mac
```
~/Library/Application Support/minecraft/resourcepacks
```

### Linux
```
~/.minecraft/resourcepacks
```
